EcoFavsManager is a free, fully customizable, and open-source tool. You can contribute to its development by visiting the LudivineCh GitHub account. If you have any questions or suggestions, feel free to open an issue on the repository.

💡 Tip: Create a bookmark folder named "sep" to add a visual separator line between your bookmarks. This works both at the top level (between bookmark cards) and inside subfolders.
